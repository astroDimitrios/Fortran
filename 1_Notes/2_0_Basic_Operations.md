# Basic Operations

## Maths

`+ - * /`    

## Character

`//` Concatenation

Slicing
`word(i:j) `    
**Including j!!!! (Starting index at 1 default)**

## Logic

````fortran
.not.
.and.
.or.
.eqv.
.neqv.
````

In order of precedence (eqv and neqv same).

## Relations

````fortran
== ! .EQ.
/= ! .NE.
<  ! .LT.
>  ! .GT.
<= ! .LE.
>= ! .GE.
````
